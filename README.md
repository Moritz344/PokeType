# PokeType
PokeType is a tui written with typescript and bun to view
information about pokemons.

![github_poketype](https://github.com/user-attachments/assets/f6f764fa-1fc0-474a-9a52-17df939d2030)


```bash
Usage: PokeType [options] [command]

a tui to get information about pokemons

Options:
  -V, --version         output the version number
  -h, --help            display help for command

Commands:
  tui                   start the tui
  search <pokemonName>  search for a specific pokemon with the name or id
  help [command]        display help for command
```
