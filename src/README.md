# PokeType
PokeType is a tui written with typescript and bun to view
information about pokemons.


```bash
Usage: PokeType [options] [command]

a tui to get information about pokemons

Options:
  -V, --version         output the version number
  -h, --help            display help for command

Commands:
  tui                   start the tui
  search <pokemonName>  search for a specific pokemon with the name or id
  help [command]        display help for command
```
